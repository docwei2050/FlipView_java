# FlipView_java
从中间翻页的翻页效果

一、本项目来源
   来自[emilsjolander/android-FlipView](https://github.com/emilsjolander/android-FlipView)
   
二、项目改造

   1.解决当item内容为图片时，翻页自动滚到90度时，图片错误的bug
   
   2.预加载3张图片
   
   3.精简拦截触摸逻辑
